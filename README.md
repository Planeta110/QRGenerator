## Instalación
Descarga el archivo del proyecto, descomprimelo despues abre una nueva terminal y escribe

`cd Downloads\QRGenerator-main\`

Despues escribe en la terminal 
```cmd
npm run dev
```
y ahora ve a tu navegador favorito y pon
`localhost:5173`.

Para la build pon :
```cmd
npm run build
```
